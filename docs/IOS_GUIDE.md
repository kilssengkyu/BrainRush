# iOS 개인 기기 테스트 가이드 (Free Provisioning)

Mac 환경이 준비되었을 때, 유료 개발자 계정($99/년) 없이 **무료 Apple ID**만으로 아이폰에서 앱을 테스트하는 방법입니다.

## 1. 사전 준비
- **Mac**: Xcode가 설치된 Mac 또는 MacBook.
- **iPhone**: USB 케이블로 Mac에 연결.
- **Apple ID**: 사용 중인 일반 Apple 계정.

## 2. Xcode 설정
1. 프로젝트 폴더에서 `npx cap open ios`를 실행하여 Xcode를 엽니다.
2. 좌측 네비게이터에서 가장 상단에 있는 **App** 프로젝트 아이콘을 클릭합니다.
3. **Targets** 목록에서 **App**을 선택합니다.
4. 상단 탭에서 **Signing & Capabilities**를 클릭합니다.
5. **Team** 드롭다운 메뉴에서 **Add an Account...**를 선택합니다.
6. 본인의 **Apple ID**로 로그인합니다.
7. 다시 **Team** 드롭다운에서 **[본인 이름] (Personal Team)**을 선택합니다.
   - *Bundle Identifier가 중복된다는 오류가 뜨면, 뒤에 숫자를 붙이거나 변경해서 고유하게 만듭니다.*

## 3. 기기 연결 및 빌드
1. 아이폰을 Mac에 케이블로 연결합니다.
2. Xcode 상단 중앙의 기기 선택 메뉴(시뮬레이터 목록)에서 연결한 **본인의 iPhone**을 선택합니다.
3. **Run** 버튼 (▶️)을 클릭합니다.

## 4. 신뢰할 수 있는 개발자 설정 (최초 1회)
앱이 설치되지만 실행되지 않고 보안 팝업이 뜰 수 있습니다. 아이폰에서 아래 설정을 진행하세요.

1. **설정 (Settings)** 앱 실행.
2. **일반 (General)** -> **VPN 및 기기 관리 (VPN & Device Management)** 선택.
3. **개발자 앱 (Developer App)** 항목에 있는 본인의 이메일을 선택.
4. **'[이메일] 신뢰함'** 버튼을 누릅니다.

## 5. 완료
이제 아이폰에서 앱 아이콘을 눌러 실행하고 테스트할 수 있습니다.

> [!NOTE]
> 무료 계정으로 설치한 앱은 **7일간 유효**합니다. 7일이 지나면 앱이 실행되지 않으며, 컴퓨터에 연결하여 다시 빌드(Run)해주면 기간이 연장됩니다.
